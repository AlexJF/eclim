Java Code Completion
====================

Java code completion uses the standard <a href="../code_completion.html">Vim
code completion mechanism</a> like so\:

::

  System.o<Ctrl-X><Ctrl-U>
  System.out.pri<Ctrl-X><Ctrl-U>
